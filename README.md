# 💳 Loan Approval Prediction using Decision Tree

## 📌 Description
This project predicts whether a loan will be approved based on income, credit score, and age using a Decision Tree Classifier.

## 🚀 Features
- Classification model
- Real-world use case
- Decision tree visualization
- Mobile-friendly (Google Colab)

## 🛠 Tech Used
- Python
- Scikit-learn
- Pandas
- Matplotlib

## 📊 Output
Predicts loan approval status (Approved/Rejected)

## 📱 Platform
Google Colab
